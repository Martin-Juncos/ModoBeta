# ModoBeta Landing Page

Landing page institucional para ModoBeta, empresa de desarrollo de software a medida.

## Tecnologías

- React
- Vite
- React Router DOM
- TypeScript
- CSS moderno
- Deploy preparado para Vercel

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

El proyecto está preparado para desplegarse en Vercel.
